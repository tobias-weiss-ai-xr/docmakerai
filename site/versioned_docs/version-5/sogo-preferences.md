# Preferences & Settings

Customize your SOGo 5 experience by adjusting personal preferences such as language, time zone, notification settings, and default views.

## Prerequisites

- A SOGo 5 account with valid credentials
- You are logged into SOGo 5

## Step-by-Step Instructions

### Step 1: Open Preferences

Click the **gear icon** ⚙ (Settings) in the top toolbar to open the Preferences page.

### Step 2: General Settings

The **General** tab allows you to configure:

| Setting | Description | Example |
|---------|-------------|---------|
| **Language** | Interface language | Deutsch, English |
| **Time Zone** | Your local time zone | Europe/Berlin |
| **Date Format** | How dates are displayed | DD.MM.YYYY |
| **Time Format** | 12-hour vs 24-hour | 24-hour format |
| **First Day of Week** | Which day starts the week | Monday |

### Step 3: Notification Preferences

Switch to the **Notifications** tab to configure:

- Email notifications for new calendar events
- Reminder emails before upcoming events
- Digest frequency (daily, weekly, or per-event)

## Configuration Options

### Interface Settings

- **Theme**: Choose between light and dark themes (if available)
- **Default Module**: Which module to show on login (Mail, Calendar, or Contacts)
- **Items per Page**: How many items to show in lists

### Calendar Defaults

| Preference | Description |
|------------|-------------|
| **Default View** | Day, Week, Month, or Year |
| **Working Hours** | Define your work day start and end times |
| **Default Event Duration** | Length of new events (30 min, 1 hour, etc.) |

## Troubleshooting

| Issue | Possible Cause | Solution |
|-------|---------------|----------|
| Settings not saving | Session timeout | Log out and back in, then try again |
| Language change not applied | Requires page reload | Refresh the page after changing the language setting |
| Calendar defaults not applying | Module-specific override | Check if your calendar has its own settings override |
