# Kontakte bearbeiten und löschen

Erfahren Sie, wie Sie Kontaktinformationen aktualisieren oder Kontakte aus Ihrem SOGo 5-Adressbuch entfernen.

## Voraussetzungen

- Ein SOGo 5-Konto mit gültigen Anmeldedaten
- Sie sind bei SOGo 5 angemeldet
- Mindestens ein vorhandener Kontakt in Ihrem Adressbuch

## Teil 1: Einen Kontakt bearbeiten

### Schritt 1: Kontakt auswählen

Klicken Sie in der linken Seitenleiste auf **Kontakte**, um das Adressbuch zu öffnen.

Suchen Sie den Kontakt, den Sie bearbeiten möchten, und klicken Sie auf seinen Namen oder Eintrag.

### Schritt 2: Kontaktdetails ändern

Der Kontakteditor ermöglicht Ihnen folgende Änderungen:

| Feld: Description | Beschreibung |
|------|-------------|
| **Vorname** | Vorname |
| **Nachname** | Familienname |
| **E-Mail** | Primäre E-Mail-Adresse |
| **Telefon** | Telefonnummer |
| **Mobil** | Mobiltelefonnummer |
| **Firma** | Organisation oder Arbeitgeber |
| **Notizen** | Freitext-Notizen zum Kontakt |

So bearbeiten Sie ein Feld:

1. Klicken Sie in das Feld, das Sie ändern möchten
2. Aktualisieren Sie den Text nach Bedarf
3. Klicken Sie auf **Speichern**, um alle Änderungen auf einmal zu übernehmen

### Schritt 3: Änderungen speichern

Klicken Sie auf die Schaltfläche **Speichern**, um Ihre Änderungen zu übernehmen. Die Kontaktdaten werden sofort aktualisiert.

## Teil 2: Einen Kontakt löschen

### Schritt 1: Kontakt öffnen

Klicken Sie auf den Kontakt, den Sie aus Ihrem Adressbuch entfernen möchten.

### Schritt 2: Löschen

1. Klicken Sie auf die Schaltfläche **Löschen** im Kontakteditor
2. Bestätigen Sie die Löschung bei Aufforderung

:::warning
Die Löschung ist endgültig. Einmal gelöscht, kann der Kontakt nicht wiederhergestellt werden.
:::

## Fehlerbehebung

| Problem: Description | Mögliche Ursache | Lösung |
|---------|-----------------|--------|
| Kontakt kann nicht bearbeitet werden | Schreibgeschütztes Adressbuch (von einem anderen Benutzer geteilt) | Sie können Kontakte in freigegebenen Adressbüchern nur anzeigen |
| Änderungen werden nicht gespeichert | Sitzung abgelaufen | Aktualisieren Sie die Seite und versuchen Sie es erneut |
| Falscher Kontakt angezeigt | Suchfilter aktiv | Löschen Sie alle Suchbegriffe in der Kontaktliste |

## Fazit

Sie haben erfolgreich gelernt, wie Sie Kontakte in Ihrem SOGo 5-Adressbuch bearbeiten und löschen.
## Accessibility

### Keyboard Navigation

This application supports keyboard navigation. No mouse required for completing this task.

| Action | Keyboard Shortcut: What key to press | Notes: Additional information |
|--------|--------------------------------------|------------------------------|
| | Navigate modules | `Tab` / `Shift+Tab` | Cycles through sections |
| | Select/activate | `Enter` or `Space` | Activate button or link |
| | Cancel/close | `Escape` | Cancel current action |
| | Navigate lists | `Arrow keys` | Move through items |

**Screen Reader Navigation Order:**
1. Sidebar navigation → `Tab` to enter
2. Module content → `Arrow keys` to navigate
3. Action buttons → `Space` or `Enter` to activate
4. Forms → `Tab` between fields, arrows for dropdowns

### High Contrast Mode

SOGo supports high contrast and dark mode. Toggle via user preferences or use browser/OS-level accessibility settings:
- **Windows:** `Win+Ctrl+C` toggles high contrast
- **macOS:** System Preferences → Accessibility → Display → Increase contrast
- **Browser Extensions:** Dark Reader, High Contrast (Chrome)
