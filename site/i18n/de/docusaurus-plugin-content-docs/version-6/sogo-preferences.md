# Einstellungen & Optionen

Passen Sie Ihre SOGo 5-Erfahrung an, indem Sie persönliche Einstellungen wie Sprache, Zeitzone, Benachrichtigungseinstellungen und Standardansichten anpassen.

## Voraussetzungen

- Ein SOGo 5-Konto mit gültigen Anmeldedaten
- Sie sind bei SOGo 5 angemeldet

## Schritt-für-Schritt-Anleitung

### Schritt 1: Einstellungen öffnen

Klicken Sie auf das **Zahnradsymbol** ⚙ (Einstellungen) in der oberen Symbolleiste, um die Einstellungsseite zu öffnen.

### Schritt 2: Allgemeine Einstellungen

Der Tab **Allgemein** ermöglicht Ihnen die Konfiguration von:

| Einstellung: Description | Beschreibung | Beispiel |
|-------------|--------------|----------|
| **Sprache** | Oberflächensprache | Deutsch, English |
| **Zeitzone** | Ihre lokale Zeitzone | Europe/Berlin |
| **Datumsformat** | Wie Daten angezeigt werden | DD.MM.YYYY |
| **Zeitformat** | 12-Stunden vs. 24-Stunden | 24-Stunden-Format |
| **Erster Wochentag** | Welcher Tag die Woche beginnt | Montag |

### Schritt 3: Benachrichtigungseinstellungen

Wechseln Sie zum Tab **Benachrichtigungen**, um zu konfigurieren:

- E-Mail-Benachrichtigungen für neue Kalenderereignisse
- Erinnerungs-E-Mails vor anstehenden Ereignissen
- Zusammenfassungshäufigkeit (täglich, wöchentlich oder pro Ereignis)

## Konfigurationsoptionen

### Oberflächeneinstellungen

- **Design**: Wählen Sie zwischen hellem und dunklem Design (falls verfügbar)
- **Standardmodul**: Welches Modul bei der Anmeldung angezeigt wird (E-Mail, Kalender oder Kontakte)
- **Elemente pro Seite**: Wie viele Elemente in Listen angezeigt werden

### Kalender-Standardwerte

| Einstellung: Description | Beschreibung |
|-------------|-------------|
| **Standardansicht** | Tag, Woche, Monat oder Jahr |
| **Arbeitszeiten** | Definieren Sie Beginn und Ende Ihres Arbeitstages |
| **Standard-Ereignisdauer** | Länge neuer Ereignisse (30 Min., 1 Stunde usw.) |

## Fehlerbehebung

| Problem: Description | Mögliche Ursache | Lösung |
|---------|-----------------|--------|
| Einstellungen werden nicht gespeichert | Sitzung abgelaufen | Melden Sie sich ab und wieder an, dann versuchen Sie es erneut |
| Sprachänderung nicht übernommen | Erfordert Seitenneuladung | Aktualisieren Sie die Seite nach der Sprachänderung |
| Kalender-Standardwerte werden nicht übernommen | Modulspezifische Überschreibung | Prüfen Sie, ob Ihr Kalender eigene Einstellungsüberschreibungen hat |

## Fazit

Sie haben erfolgreich gelernt, wie Sie Ihre SOGo 5-Einstellungen anpassen.
## Accessibility

### Keyboard Navigation

This application supports keyboard navigation. No mouse required for completing this task.

| Action | Keyboard Shortcut: What key to press | Notes: Additional information |
|--------|--------------------------------------|------------------------------|
| | Navigate modules | `Tab` / `Shift+Tab` | Cycles through sections |
| | Select/activate | `Enter` or `Space` | Activate button or link |
| | Cancel/close | `Escape` | Cancel current action |
| | Navigate lists | `Arrow keys` | Move through items |

**Screen Reader Navigation Order:**
1. Sidebar navigation → `Tab` to enter
2. Module content → `Arrow keys` to navigate
3. Action buttons → `Space` or `Enter` to activate
4. Forms → `Tab` between fields, arrows for dropdowns

### High Contrast Mode

SOGo supports high contrast and dark mode. Toggle via user preferences or use browser/OS-level accessibility settings:
- **Windows:** `Win+Ctrl+C` toggles high contrast
- **macOS:** System Preferences → Accessibility → Display → Increase contrast
- **Browser Extensions:** Dark Reader, High Contrast (Chrome)
