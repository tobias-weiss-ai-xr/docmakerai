Abmelden

Beenden Sie Ihre SOGo 5-Sitzung sicher, indem Sie sich abmelden, wenn Sie Ihre Arbeit abgeschlossen haben.

Voraussetzungen​

  • Sie sind bei SOGo 5 angemeldet

Schritt-für-Schritt-Anleitung​

Schritt 1: Abmelde-Button finden​

In der Symbolleiste oben rechts finden Sie das Ein/Aus-Symbol ⏻ (Abmelden).

Schritt 2: Zum Abmelden klicken​

Klicken Sie auf das Ein/Aus-Symbol, um Ihre Sitzung zu beenden. Sie werden zur Anmeldeseite weitergeleitet.

Schritt 3: Bestätigen, dass Sie abgemeldet sind​

Nach der Weiterleitung sehen Sie den SOGo-Anmeldebildschirm, der bestätigt, dass Ihre Sitzung sicher beendet wurde.

tipp

Wenn Sie einen gemeinsam genutzten oder öffentlichen Computer verwenden, melden Sie sich immer ab, wenn Sie fertig sind. Schließen Sie nicht einfach den Browser-Tab.

Fehlerbehebung​

Problem: DescriptionMögliche UrsacheLösung
Abmelde-Button nicht sichtbarBildschirmauflösung zu schmalVersuchen Sie, das Browserfenster zu verbreitern oder klicken Sie zuerst auf das Menüsymbol (☰)
Sitzung nach Abmeldung noch aktivBrowser hat die Seite zwischengespeichertLeeren Sie den Browser-Cache und schließen Sie alle SOGo-Tabs

Fazit​

Sie haben sich erfolgreich von SOGo 5 abgemeldet. Ihre Sitzung wurde sicher beendet.

Accessibility​

Keyboard Navigation​

This application supports keyboard navigation. No mouse required for completing this task.

ActionKeyboard Shortcut: What key to pressNotes: Additional information
Navigate modulesTab / Shift+Tab
Select/activateEnter or Space
Cancel/closeEscape
Navigate listsArrow keys

Screen Reader Navigation Order:

  1. Sidebar navigation → Tab to enter
  2. Module content → Arrow keys to navigate
  3. Action buttons → Space or Enter to activate
  4. Forms → Tab between fields, arrows for dropdowns

High Contrast Mode​

SOGo supports high contrast and dark mode. Toggle via user preferences or use browser/OS-level accessibility settings:

  • Windows: Win+Ctrl+C toggles high contrast
  • macOS: System Preferences → Accessibility → Display → Increase contrast
  • Browser Extensions: Dark Reader, High Contrast (Chrome)
