Mail — Folder Management

Organize your emails using folders for archiving and easy access.

Prerequisites​

  • A SOGo 5 account with valid credentials
  • You are logged into SOGo 5

Step-by-Step Instructions​

Step 1: Open the Mail Module​

In the sidebar, click Mail to open the inbox. The folder list appears on the left side of the email view.

Step 2: Browse Folders​

The folder list shows your default folders:

  • Inbox — Receives incoming emails
  • Sent — Contains emails you have sent
  • Drafts — Stores draft emails not yet sent
  • Trash — Holds deleted emails
  • Spam — Contains messages identified as spam

Step 3: Create a New Folder​

  1. Right-click in the folder list area
  2. Select New Folder from the context menu
  3. Enter a folder name
  4. Click Create

Step 4: Move Emails Between Folders​

Method 1: Drag and Drop​

  1. Click the email you want to move
  2. Drag it to the target folder in the list
  3. Release to drop

Method 2: Using the Move Button​

  1. Select an email
  2. Click the Move button (folder icon with arrow)
  3. Select the target folder from the dropdown
info

Folder changes are synchronized with the server immediately. Your folder structure will be consistent across devices.

Folder Actions​

Action: DescriptionHow To
RenameRight-click folder → Rename
DeleteRight-click folder → Delete (cannot delete default folders like Inbox)
Empty TrashRight-click Trash → Empty Trash

Accessibility​

Keyboard Navigation​

This application supports keyboard navigation. No mouse required for completing this task.

ActionKeyboard Shortcut: What key to pressNotes: Additional information
Navigate modulesTab / Shift+Tab
Select/activateEnter or Space
Cancel/closeEscape
Navigate listsArrow keys

Screen Reader Navigation Order:

  1. Sidebar navigation → Tab to enter
  2. Module content → Arrow keys to navigate
  3. Action buttons → Space or Enter to activate
  4. Forms → Tab between fields, arrows for dropdowns

High Contrast Mode​

SOGo supports high contrast and dark mode. Toggle via user preferences or use browser/OS-level accessibility settings:

  • Windows: Win+Ctrl+C toggles high contrast
  • macOS: System Preferences → Accessibility → Display → Increase contrast
  • Browser Extensions: Dark Reader, High Contrast (Chrome)
